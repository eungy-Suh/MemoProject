# MemoProject

## SimpleMemo


+ 메모장에 기록을 하고 저장하여 데이터가 날아가지 않으며
+ 각 메모는 수정 및 삭제, 전체삭제가 가능하고
+ 해야할 일을 다 했으면 취소선을 넣을 수 있으며
+ 감추고 싶은 메모가 있으면 비밀번호를 입력해야 들어갈수있는, 프라이버시를 지킬 수 있습니다.
